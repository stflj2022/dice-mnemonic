# 骰子助记词生成器 (Dice Mnemonic Generator)

基于物理骰子的 BIP39 助记词生成器，完全离线运行。

## 特性

- 100% 离线运行（使用系统字体，无网络请求）
- 物理骰子提供真随机熵
- 鼠标移动熵收集（增强随机性）
- 页面关闭时内存擦除（防止数据残留）
- 独立验证按钮（可随时验证助记词校验和）

## 使用方法

1. 在浏览器中打开 `index.html`
2. 选择助记词长度（12词或24词）
3. 通过快速点击按钮或输入数字记录掷骰结果
4. 达到足够次数后自动生成 BIP39 助记词
5. 点击验证按钮确认校验和
6. 手动抄写助记词（不要截图或保存为数字文件）

## 安全提示

- 请在离线设备上使用
- 生成完毕后请用纸和笔抄写助记词
- 掷骰过程建议独自完成

## 技术说明

- 1颗骰子 = log₂(6) ≈ 2.585 bit
- 12词 = 128bit → 需掷 ≥ 50次
- 24词 = 256bit → 需掷 ≥ 99次
- 方法：将点数序列哈希 → 截取熵 → BIP39映射
